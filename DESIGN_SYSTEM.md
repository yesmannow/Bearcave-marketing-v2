# UI/UX EXECUTIVE AGENT: THE DESIGN GUARDIAN

## 1. The 'Professional Restraint' Mandate
- **Goal:** Achieve a 'Global Agency' aesthetic (Clay, ToyFight).
- **Rule of Simplicity:** If a design element (glow, animation, or 3D effect) does not serve a functional or narrative purpose, it must be removed.
- **Typography First:** Design begins with type. Use the established 'OLED White' (`#EDEDED`) and ensure line-heights and letter-spacing are optimized for executive readability.

## 2. The 'Ocean Pearl' Design System
- **Palette:** Teal (`#40E0D0`) for System Success; Orange (`#FFA500`) for Strategic Call-to-Actions.
- **Surface Logic:** Use `ocean-pearl-glass` (12px blur, 0.7 opacity) for all floating panels.
- **Tactile Feedback:** Every button must use the `pearl-interact` class to provide a subtle 2px lift and soft glow on hover.

## 3. UX Conversion Logic
- **The 'Promise' (Hero):** Is the value proposition clear within 3 seconds?
- **The 'Proof' (Work):** Are case studies linked to hard metrics? Use Orange accents for numbers ($ REV, % GROWTH).
- **The 'Path' (CTA):** The 'Book Discovery' button must be the most visually distinct element on any page.

## 4. Workstation Specialization (Piko Project)
- **Native Fidelity:** The [Piko Studio](https://piko-artist-website.vercel.app/studio) must feel like a high-end DJ tool.
- **Optimization:** Use `will-change: transform` on the Mixer and Decks. Ensure no layout shifts occur during audio playback.
